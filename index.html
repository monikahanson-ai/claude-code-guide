01 — Introduktion

Vad är Claude Code?

Claude Code är ett agentic kodverktyg från Anthropic. Det körs i din terminal och ger Claude direkt tillgång till din kodbas, dina filer och ditt system.

Inte en plugin

Claude Code är ett CLI-verktyg — det körs i terminalen, inte som ett tillägg i din editor. Det ger full systemåtkomst och kan läsa, skriva och köra kod på riktigt.

Förstår hela projektet

Till skillnad från en chatbot läser Claude Code hela din kodbas och förstår sammanhang, beroenden och arkitektur innan det ger förslag.

Agentisk — handlar självständigt

Claude kan ta flera steg i följd: söka efter filer, redigera dem, köra tester och iterera — utan att du behöver guida varje steg.

Du behåller kontrollen

Varje fil-ändring och kommando visas tydligt. Du kan granska, godkänna eller avbryta vad som helst. Claude Code ber alltid om lov innan något känsligt görs.

02 — Flöde

Hur det fungerar

Claude Code arbetar i en interaktiv loop direkt i din terminal.

01

Du startar Claude i terminalen

Navigera till ditt projekt och kör claude. Verktyget läser automatiskt in projektets filer och skapar kontext.

$ cd mitt-projekt && claude
02

Du ger en uppgift i klartext

Skriv vad du vill ha gjort på vanlig svenska eller engelska. Inga special-kommandon behövs — Claude förstår naturligt språk.

Lägg till felhantering i alla API-anrop i src/api/
03

Claude planerar och agerar

Claude analyserar koden, skapar en plan och börjar arbeta. Det visar varje steg — vilka filer som öppnas, ändras eller körs — och ber om godkännande vid behov.

04

Du granskar och itererar

Se ändringarna som en diff. Godkänn, avvisa eller be Claude justera. Processen upprepas tills resultatet är precis som du vill ha det.

03 — Kapaciteter

Vad kan Claude Code göra?

Från rutinuppgifter till komplexa arkitekturproblem.

Kodning

Skriver ny kod

Genererar funktioner, klasser, komponenter och hela moduler baserat på din beskrivning och befintlig kodstil.

Felsökning

Hittar och fixar buggar

Analyserar felmeddelanden, spårar ursprunget och föreslår—eller implementerar direkt—korrekta lösningar.

Refaktorering

Förbättrar befintlig kod

Identifierar kodlukt, duplicering och antipatterns. Refaktorerar säkert med hänsyn till hela kodbasens sammanhang.

Testning

Skriver och kör tester

Genererar enhetstester, integrationstester och edge cases. Kan köra testerna och fixa de som fallerar.

Dokumentation

Dokumenterar kod

Skriver JSDoc, docstrings, README-filer och arkitektur-beskrivningar baserat på den faktiska koden.

Git

Hanterar versionshantering

Skapar commits med beskrivande meddelanden, granskar diff:ar och hjälper till med merge-konflikter.
